# vue-easytable

[切换中文](https://github.com/huangshuwei/vue-easytable/blob/dev/README-CN.md)


## Introduction
Based on vue2.x flexible table components.

## Features

- Easy to use
- Powerful
- Unit test coverage greater than 80%

## API & Examples
[官方文档](http://doc.huangsw.com/vue-easytable)

## Supports

- [x] Internationalization
- [x] Theme Custom
- [x] Built in theme
- [x] Column Fixed
- [x] Header Fixed
- [x] Header Grouping
- [x] Filter
- [x] Sort
- [x] Cell Style
- [x] Cell Span
- [x] Cell Custom
- [x] Cell Selection(keyboard operation)
- [x] Cell Ellipsis
- [x] Row Radio
- [x] Row Checkbox
- [x] Row Expand
- [x] Row Style
- [x] Footer Summary
- [x] Pagination
- [x] Virtual Scroll
- [x] Event Custom
- [x] Loading component
