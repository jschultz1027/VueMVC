# vue-easytable

## Introduction
一个基于 Vue2.x 的表格组件
## 特点

- 使用简单
- 功能强大
- 单元测试覆盖率大于 80%

## API & Examples
[官方文档](http://doc.huangsw.com/vue-easytable)

## Supports

- [x] 国际化
- [x] 主题定制
- [x] 内置主题
- [x] 固定列
- [x] 固定表头
- [x] 表头分组
- [x] 筛选
- [x] 排序
- [x] 单元格样式
- [x] 单元格合并
- [x] 单元格自定义
- [x] 单元格选择（键盘操作）
- [x] 单元格省略
- [x] 行单选
- [x] 行多选
- [x] 行展开
- [x] 行样式
- [x] footer
- [x] 分页
- [x] 虚拟滚动
- [x] 自定义事件
- [x] 内置loading
